# Koo Fruit Engine — Documentation

Public user manual and documentation for the **Koo Fruit Engine** —
a Blender 4.2+ extension for photo-driven realistic banana generation.

- 📖 **Manual (HTML)**: [`user_manual.html`](user_manual.html) — best viewed via GitHub Pages
- 📄 **Manual (PDF)**: [`user_manual.pdf`](user_manual.pdf) — 29 pages with screenshots
- 📝 **Manual (Markdown source)**: [`user_manual.md`](user_manual.md)

## Where to get the addon

Commercial release on Superhive (Blender Market). This repository hosts
documentation only — the addon source is maintained in a separate
private repository and distributed via the marketplace.

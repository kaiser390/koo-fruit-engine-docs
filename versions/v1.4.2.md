## [1.4.2] — 2026-04-26

LOW-quality Realistic Generate **6.08 s → ~2.05 s (≈3× faster)**.
Photo→Stylized converter, slimmer parameter UI, surface bump
controls, build-include checkboxes.

### Added

- **Photo → Stylized (keep skin)** — measure a photo-marker
  silhouette and copy the geometry into the Stylized sliders
  (length / radius / curvature / shoulder / curl peak / sharpness).
  Photo strip texture stays as the peel skin — edit shape with
  sliders without losing the photo-accurate colour.
- **Build-include checkboxes** above the Generate button (Stylized
  tab): Interior / Pulp / Membrane gate primitive *building*, not
  just visibility — unchecked → not generated at all (faster
  Generate, smaller scene, cleaner FBX). Falls out for free:
  peel-only / pulp-only banana variants from the same engine.
- **Surface bump** controls (Bump Amount / Scale / Micro Noise) +
  Seed Density slider exposed in the Stylized advanced panel.

### Changed

- Stylized Color section: palette dropdown is the hero control;
  Override raw colours unlocks per-layer pickers in a 2-col grid
  (Outer / Inner, Tip + Reach, Pulp / Warm, Membrane / Spot).
- Peel Geometry sliders now read as **% of body radius** /
  **% of peel thickness** instead of raw meters — banana-size
  invariant, the same value reads correctly across cultivars.
- Cavendish archetype defaults bumped to a usable neutral-banana
  baseline (length 0.22 / radius 0.030 / curvature 1.0 /
  shoulder_frac 0.33 / sharpness 0.33).
- Random Generation now rolls every advanced slider including the
  newly surfaced peel + surface params. Bundle count + thickness
  intentionally NOT randomised (quality choices, not visual
  identity).

### Fixed

- Realistic↔Stylized round-trip: Generate dispatch follows the
  `meta.workflow_mode` tab rather than marker presence — switching
  tabs no longer demands "Confirm markers first".
- Strip-mode UV sampling reads `fe_rest_co` so the strip stays
  aligned after the peel curl bake.

### Hidden

- Four Color sliders (`ripeness`, `saturation_scale`, `spot_density`,
  `honey_band_intensity`) hidden in the panel pending consumer
  hookup. Properties remain in the JSON schema — existing presets
  round-trip unchanged.
